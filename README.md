# family-tree-management
Family Tree Management System là ứng dụng web giúp lưu trữ, quản lý và trực quan hóa gia phả gia đình. Hệ thống cho phép đăng ký, đăng nhập, quản lý thông tin thành viên, hiển thị sơ đồ gia phả dạng cây, đồng thời hỗ trợ lưu album ảnh kèm địa điểm và quản lý các sự kiện gia đình trên lịch.
